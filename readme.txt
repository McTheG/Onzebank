Dit project bestaat uit 2 kleine projecten. Deze zijn te onderscheiden door te kijken naar de mappen.
De map onzebank bevat een applicatie waarop iedereen zou moeten kunnen komen. Deze applicatie kent geen beveiliging en ook geen inlog mogelijkheden.
De map onzebank-sec bevat bijna dezelfde applicatie alleen deze behoord alleen beschikbaar te zijn via een https verbinding. Ook kent de applicatie een inlog procedure.

Om de pagina's van de website te vinden moet je in de map pages zijn. Hierin staat per pagina een bestand met als inhoud een object die getekend word op het schem via de functie draw().
verder zijn de emails opgenomen via een template parser. Deze zijn dan ook te vinden in de map templates. De map mose bevat de website engine.

Voor het installeren van de applicatie moet het bestand config.php aangepast worden. In dit bestand staan de gegevens voor mysql en andere modules.
